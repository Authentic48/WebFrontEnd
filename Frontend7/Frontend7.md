# Practice 7
## Objective
1. Learn how to search elements on a page.
2. Learn how to change values of properties of DOM elements.
3. Figure out the DOM events system, learn how to subscribe to event notifications and process them.

## Result
Develop a simple calculator using JavaScript. The corresponding HTML – layout and CSS are given in the template of the task, making them on your own is unnecessary.
